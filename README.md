# Codsoft-data-science-internship
This repository contains all tasks completed during my Data Science Internship at CodSoft, covering projects like Titanic Survival Prediction, Iris Classification, and Credit Card Fraud Detection using machine learning techniques.



## 🚢 Titanic Survival Prediction (Classification Project)

### 📌 Project Overview

This project predicts whether a passenger survived or not using machine learning classification algorithms.

---

### ⚙️ Techniques Used

* Data Cleaning & Preprocessing
* Handling Missing Values
* Encoding Categorical Variables
* Train-Test Split
* Feature Scaling (for Logistic Regression)

---

### 🤖 Models Implemented

* Logistic Regression
* Decision Tree
* Random Forest

---

### 📊 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

---

### 📈 Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 79.88%   |
| Decision Tree       | 77.65%   |
| Random Forest       | 81.56%   |

---

### 🏆 Conclusion

Random Forest performed the best among all models, providing the highest accuracy and balanced predictions.

---

### 🚀 Key Learnings

* Importance of data preprocessing
* Difference between classification models
* Model evaluation using multiple metrics
* Comparing models to select the best one

---

### 📁 Dataset

Titanic dataset from Kaggle
